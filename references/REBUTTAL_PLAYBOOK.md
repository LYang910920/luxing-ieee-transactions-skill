# Rebuttal Playbook

## Comment classification

Tag each comment as:

```text
validity
novelty
clarity
missing_experiment
fairness
proof
implementation
presentation
citation
scope
editorial
```

## Response structure

1. Thank and restate the issue precisely.
2. State whether you agree, partially agree, or clarify a misunderstanding.
3. Give the action and evidence.
4. Quote only the minimum revised text needed.
5. Give exact page/section/line or equation/figure/table location.
6. State any remaining limitation honestly.

## Strong response pattern

```text
We agree that the original manuscript did not establish [X]. We therefore
(1) narrowed Claim 2 from [broad] to [scoped],
(2) added [baseline/check/proof] in Section Y,
(3) reported [metric and uncertainty] in Table Z, and
(4) added a limitation covering [remaining boundary].
```

## Rules

- Never say “done” without a manuscript location.
- Do not defend an avoidable ambiguity; revise it.
- Do not use a new experiment to conceal a theorem or modeling flaw.
- Match every rebuttal claim to the revised files and rerun the relevant analysis.
- Keep tone factual and non-adversarial.
