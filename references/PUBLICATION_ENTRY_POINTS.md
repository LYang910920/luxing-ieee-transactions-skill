# Publication Entry Points

Use these as starting points, then reconcile with DOI and publisher metadata:

```text
Deakin Experts profile:
https://experts.deakin.edu.au/43777-luxing-yang

Deakin Experts publications:
https://experts.deakin.edu.au/43777-luxing-yang/publications

Google Scholar author profile:
https://scholar.google.com/citations?user=Nhd2-XEAAAAJ

ORCID:
https://orcid.org/0000-0002-9229-5787

DBLP:
https://dblp.org/pid/59/10736
```

## Reconciliation order

1. Normalize title punctuation, hyphens and capitalization.
2. Normalize author names (`Lu-Xing Yang`, `L.-X. Yang`, `Luxing Yang`).
3. Prefer DOI as the record key when verified.
4. Resolve online-first versus issue year explicitly.
5. Mark errata, editorials, reviews, conference versions and duplicate preprints.
6. Record index disagreements and the selected canonical metadata source.

Do not rely on citation counts or current affiliation without recording the retrieval date; those fields are time-sensitive and not required for style calibration.
